Экзаменационные задания

1. Сделать эффект шлейфа: при движении круга за ним остаются несколько кругов в предыдущих положениях курсора (размер и цвет менять необязательно). Хранить их координаты можно в массиве фиксированной длины (длина "хвоста")

2. Реализовать перетаскивание двух прямоугольников мышью. Если прямоугольники пересекаются, то выводить на экран площадь их пересечения, иначе сообщать, что они не пересекаются

При этом необходимо написать такие функции:

boolean isPointInRect(float pointX, float pointY, float rectX, float rectY, int rectWidth, int rectHeight) // возвращает true, если точка point находится внутри прямоугольника rect
        
float getIntersectionArea(float rectAX, float rectAY, int rectAWidth, int rectAHeight, float rectBX, float rectBY, int rectBWidth, int rectBHeight) // возвращает площадь пересечения прямоугольников rectA и rectB

Логические операторы:

&& // логическое И
|| // логическое ИЛИ
! // логическое НЕ
