Умный дом

Устройства, входящие в систему умного дома, могут находиться в определенной зависимости друг от друга. Например, медиа-система не может полноценно функционировать без колонок и медиа-сервера. Медиа-сервер, в свою очередь, может нуждаться в хранилище данных, содержащем медиа-контент. Для обновления библиотеки музыки этому хранилищу требуется интернет, который обеспечивается маршрутизатором. Но маршрутизатор обслуживает не только хранилище данных, но и очень много других устройств.

При выходе из строя одного из устройств, в нерабочее состояние по цепочке переходят и все зависимые от него приборы.

В описанной ситуации при поломке хранилища данных перестанет работать медиа-сервер, а за ним и вся медиа-система. Но работа колонок не имеет обратной зависимости от медиа-системы, поэтому они из строя не выйдут. Так же как и маршрутизатор, обеспечивающий доступ в интернет. Таким образом, выход из строя хранилища данных слабо сказался бы на работе всей системы в целом. Другое дело ­– поломка маршрутизатора. От него зависит большое количество устройств, работающих под управлением умного дома, так что такая поломка была бы намного критичнее.

Вам даны устройства, которые непосредственно зависят от данного. Ваша задача для каждого устройства определить все устройства, которые выйдут из строя вместе с ним при его поломке.

Входные данные:

int[][] dependents = {{2}, {2}, {}, {0}, {}, {}, {3, 5, 7}, {}};
int[][] chainingDependents = new int[8][];

Ожидаемый результат выполнения: масссив chainingDependents равный

{{2}, {2}, {}, {0, 2}, {}, {}, {3, 0, 2, 5, 7}, {}}

Для решения задачи в числе прочего необходимо будет реализовавать функции:

boolean doesArrayContainElement(int[] array, int element) // определяет, входит ли element в array
int[] findDependents(int gadget, int[] array) // рекурсивно ищет все зависимые от gadget устройства, сохраняя их в массив array

Короткого способа вывести двумерный массив в консоль нет, поэтому содержимое придется выводить с помощью цикла похожим кодом:

int[][] matrix = {
  {5, 7, 6}, 
  {31, 4, 7}, 
  {52, 51, 7}
};

print("{");
for (int[] nums : matrix) {
  print("{");
  for (int num : nums) {
    print(num + ", ");
  }
  print("}, ");
}
print("}");
